## What Run

* Go
* gin
* gorm
* Docker
* mysql

## Set Up

### 1. Install Docker
`https://docs.docker.com/`

### 2. Pull Repository
`git clone git@github.com:holefillingco-ltd/Shupple-api.git`

### 3. Docker-compose build
`docker-compose build`

### 4. Docker-compose up
`docker-compose up`
